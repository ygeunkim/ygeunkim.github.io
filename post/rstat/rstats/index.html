Hashtags appearing with rstats

Note that searchtwit is a list.

  1. Find any non-whitespace pattern after a hashtag.

  2. Remove elements #rstats.

    other_tag <- 
lapply(
searchtwit,
function(x) {
  str_extract_all(
    x$text,
    pattern = "#\\S+"
  ) %>% 
    unlist() %>% 
    str_remove_all(pattern = "[:punct:]$") %>% 
    .[str_to_lower(.) != "#rstats"]
}
) %>% 
unlist()
