# Factor Specification

**\[experimental\]** Set factor size and prior for factor.

## Usage

``` r
set_factor(size_factor = 0, factor_lag = 0)
```

## Arguments

- size_factor:

  Factor dimension. If `0`, factor is not used.

- factor_lag:

  Lag for the factor autoregressions.

## References

Korobilis, D. (2022). A new algorithm for structural restrictions in
Bayesian vector autoregressions. European Economic Review, 148, 104241.
