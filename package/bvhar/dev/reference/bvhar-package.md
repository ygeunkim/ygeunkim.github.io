# bvhar: Bayesian Vector Heterogeneous Autoregressive Modeling

Tools to model and forecast multivariate time series including Bayesian
Vector heterogeneous autoregressive (VHAR) model by Kim & Baek (2023)
([doi:10.1080/00949655.2023.2281644](https://doi.org/10.1080/00949655.2023.2281644)
). 'bvhar' can model Vector Autoregressive (VAR), VHAR, Bayesian VAR
(BVAR), and Bayesian VHAR (BVHAR) models.

## Details

The bvhar package provides function to analyze and forecast multivariate
time series data via vector autoregressive modeling. Here, vector
autoregressive modelling includes:

- Vector autoregressive (VAR) model: [`var_lm()`](var_lm.md)

- Vector heterogeneous autoregressive (VHAR) model:
  [`vhar_lm()`](vhar_lm.md)

- Bayesian VAR (BVAR) model: [`var_bayes()`](var_bayes.md)

- Bayesian VHAR (BVHAR) model: [`vhar_bayes()`](vhar_bayes.md)
